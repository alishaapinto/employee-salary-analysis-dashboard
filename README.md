# Employee Salary Analysis Dashboard

## Project Overview

This project is an Excel-based Employee Salary Analysis Dashboard created to analyze employee salary data and generate useful business insights.

The dashboard uses Pivot Tables, Pivot Charts, and KPI metrics to help understand employee distribution, salary trends, bonus allocation, and workforce demographics.

---

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Analysis
- Dashboard Design

---

## Dataset Information

The dataset contains:

- Employee ID
- Employee Name
- Department
- Gender
- Age
- Experience (Years)
- Salary
- Bonus
- Location

Total Records: 20 Employees

---

## Key Performance Indicators (KPIs)

- Total Employees
- Average Salary
- Highest Salary
- Lowest Salary

---

## Dashboard Insights

### Employee Count by Department
Shows the number of employees in each department.

### Average Salary by Department
Compares average salary across departments.

### Total Bonus by Department
Displays bonus distribution among departments.

### Employee Count by Location
Shows employee distribution across locations.

### Average Salary by Location
Compares salary trends across cities.

### Gender Distribution
Displays male and female employee count.
